MULTI-MATERIAL LINE SUPPORT (CRITICAL)

Each JW must support multiple material lines.

Add “Add Material Line” button.

Each material line contains:

Material (Dropdown + Custom)

Thickness (mm) (Optional — mandatory except for Zinc Received)

Our Weight (kg) (Mandatory)

Zinc (Auto-calculated)

CALCULATION ENGINE
Zinc Consumed Formula

Zinc Consumed = Our Weight × Rate %

Rate varies by:

Thickness

Vendor

Party Weight does NOT affect calculation.

Our Weight is final weight used for zinc % calculation.

Precision: configurable decimals.
Rounding rules: configurable.

SPECIAL MATERIAL: “ZINC RECEIVED”

If Material = “Zinc Received”:

Behavior:

Zinc value entered manually OR calculated as 100% of weight.

System treats this value as NEGATIVE consumption.

This subtracts from zinc consumed balance.

ZINC BALANCE LOGIC

Maintain vendor-wise running zinc ledger.

Zinc Ledger Formula:

Total Zinc Consumed
– Zinc Received
= Net Zinc Balance

Interpretation:

If Net Balance > 0 → Company must send more zinc to job worker.
If Net Balance < 0 → Job worker holds excess zinc.

System must show:

Running Zinc Balance

Monthly surplus summary

Vendor-wise zinc statement
